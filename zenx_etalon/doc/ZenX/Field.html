			$fname
						(line 27)
		
	


	

$fname variable contains field name.


Field name is a unique (withing a table) identifier of the field, this value is used to name  table fields within data storage engines. In order to prevent data storage engine specific  problems it is recommended to use Latin characters as a field name.

Field class constructor.

getProp() method returns Field object property specified as parameter.
