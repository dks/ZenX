createStorage (line 166)

createStorage() method performs all necessary routines to create tables from MainEngineAbstract::$tables array.

This is utility method for automatic creation of mysql tables. Tables are created considering the following parameters: 'mysqlPrefix' parameter at MainEngineAbstract::$ops is the string to prepend table name. 'imageFolder' parameter at MainEngineAbstract::$ops is the name of folder storing images, must have read/write access. 'fileFolder' parameter at MainEngineAbstract::$ops is the name of folder storing files (other than images), must have read/write access. 'mysqlMultiSuffix' parameter at MainEngineAbstract::$ops is the suffix used to form multiple values tables (referenced values).

Data charset is set to UTF-8. If image/file folder can not be accessed MySqlNoFolderCreateRightsException is thrown.

Please note, that current implementation stores files/images externally in file system instead of database, image tables contain only reference numbers for file names identification.

  • see: StorageEngineMysql::$storageMultiTypes
  • see: StorageEngineMysql::$storageDataTypes
  • see: MainEngineAbstract::setParameters()
  • throws: MySqlNoFolderCreateRightsException
